# `os`
![](os.png)

This module displays an icon of the operating system. If the operating system is Linux, bronze detects the distribution by reading the `/etc/os-release` file.

The recognized operating systems are:
* macOS
* Arch Linux
* CentOS
* Debian
* Fedora
* Linux Mint
* SUSE
* openSUSE
* Ubuntu
* ElementaryOS
